# Employee Sentiment Analysis

## Overview
This project analyzes employee email communication to assess sentiment,
engagement, and potential flight risk using NLP and statistical analysis.

## Key Tasks Completed
- Sentiment labeling (Positive, Negative, Neutral)
- Exploratory Data Analysis (EDA)
- Monthly employee sentiment scoring
- Employee ranking (Top positive & negative)
- Flight risk identification using rolling 30-day window
- Predictive modeling using linear regression

## Key Findings
- Identified employees with consistently negative sentiment
- Flagged flight-risk employees based on behavior patterns
- Observed sentiment trends over time
